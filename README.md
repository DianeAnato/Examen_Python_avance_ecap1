# Examen_Python_avance_ecap1

# Tableau de bord des ventes du supermarché

## Description
Application interactive développée avec Dash et Plotly pour analyser les ventes d’un supermarché.

## Fonctionnalités
- Filtres par sexe et ville
- KPI dynamiques
- Histogramme des montants
- Diagramme en barres
- Courbe d’évolution des ventes

## Technologies utilisées
- Python
- Dash
- Plotly Express
- Pandas

python exam.py
